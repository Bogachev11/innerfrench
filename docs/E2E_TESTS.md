# E2E-тесты (Playwright)

Перед деплоем на прод **обязательно** запускать e2e-тесты.

## Как запустить

1. В одном терминале запустить приложение:
   ```bash
   npm run dev
   ```
   Дождаться сообщения, что сервер слушает (порт 3010).

2. В другом терминале запустить тесты:
   ```bash
   npm run test:e2e
   ```

Если dev уже запущен, второй шаг можно выполнять многократно.

## Что проверяют тесты

- Страница `/vocab` (Words) загружается.
- Отображается блок статистики (Total, Due и т.д.).
- Нет пустой/сломанной страницы.

После успешного прохождения тестов можно деплоить на прод.
